Lumenis FemTouch

FemTouch — модуль для проведения процедур вульво-вагинального омоложения.

Модуль FemTouch лазера AcuPulse предназначен для фракционной абляции внутренних стенок влагалища и вульвы. Используется для лечения генитоуринарного синдрома, вульвовагинального омоложения, стрессового недержания мочи, опущения стенок влагалища.
